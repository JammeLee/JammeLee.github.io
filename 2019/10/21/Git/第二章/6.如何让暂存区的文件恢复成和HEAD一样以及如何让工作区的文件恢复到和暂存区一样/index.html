如何让暂存区的文件恢复到和HEAD一样以及如何让工作区的文件恢复到和暂存区一样

如何让暂存区的文件恢复到和HEAD一样

  • git reset HEAD

如何取消暂存区部分文件的修改

  • git reset HEAD —[filename1,filename2,…]

如何让工作区的文件恢复到和暂存区一样

  • git checkout — [filename]
